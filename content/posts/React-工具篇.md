## Babel

js 编译器，AST -> Transform -> Generate

功能：

1. ES6 转 ES5 语法(@babel/core + @babel/preset-env)
2. Polyfill : 引入额外代码，使新功能可以在旧的浏览器中使用
3. JSX 转 JS (@babel/preset-react)
4. 插件 (plugins)

## Swc

速度块

功能：

1. 编译、转换（@swc/core）
2. 打包
