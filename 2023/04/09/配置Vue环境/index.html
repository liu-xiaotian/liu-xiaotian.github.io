配置Vue环境

  1. 下载nodejs安装包，安装完成

  2. 检验是否下载完成。
    npm -v
    node -v

  3. 安装vue的脚手架

    • Vue2.9.6: npm install -g vue-cli
    • Vue5.x : npm install -g @vue/cli

  4. vue -V

  5. 安装项目

    • Vue2.9.6

    • Vue5.x

  6. 运行

    • vue2.9.6: npm run dev
    • Vue5.x : npm run ser

安装scss

注意：

  1. 文件夹命名不要采用中文
  2. 安装文件目录和项目目录需要分清晰，不要混合在一起
  3. npm 的镜像地址需要改变
  4. 在本地部署项目
    检查是不是有node_modules文件夹
    如果没有就npm install
