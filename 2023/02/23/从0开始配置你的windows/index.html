从0开始配置你的windows

从0开始配置你的windows

给电脑重新安装了系统，记录一下配置的过程

  1. 在登陆微软账号时可以选择脱机使用（离线），如果登陆了账号后面的配置会变慢，并且用户名默认是账号的前5位。
  2. 进入桌面首先联网，然后更新驱动。
  3. 这时再登陆微软账号
  4. 设置桌面图标，任务栏图标
  5. 打开文件管理器，设置打开显示此电脑，显示文件拓展名，显示隐藏文件

首先解决网络问题，clash懂的都懂。

一台电脑最重要的软件莫过于浏览器，有了浏览器连上网，我们就可以访问大千世界。

我的主力浏览器 是google浏览器，因此想要同步配置就需要科学上网。

  • 网盘：百度网盘、阿里云盘
  • 网速查看：TrafficMonitor()
  • 效率神器：Qucker
  • 文件搜索：Everything
  • 局域网传输：LocalSend
  • 桌面整理：Fences
  • 视频播放器：PotPlayer
  • 截图工具：Snipaste
  • 剪贴板：Ditto
  • 写文档：typora
  • 图床工具：Picgo
  • 软件卸载：lObit
  • 网盘挂载：alist
