nvm

发表于 2023-03-09 更新于 2023-04-23

  • nvm

    NVM：Node Version Manage，即Node的版本管理工具。使用NVM，可以很方便地在多个NodeJS版本之间进行切换。

    一、下载安装

    1. 下载地址：
      GitHub选择下载nvm-setup.zip

    2. 注意安装路径：第一个路径就是软件安装路径，第二个选择NodeJS的Symlink (符号链接)文件夹的位置，用于生成NodeJS的映射目录，路径+\nodejs ,这个文件夹需要提前创建，如果直接填写路径可能不会自动生成，后面也需要新建）

    3. 更改setting中的配置（淘宝镜像)
      node_mirror: https://npm.taobao.org/mirrors/node/
      npm_mirror: https://npm.taobao.org/mirrors/npm/

    二、使用

    • nvm list 查看安装的所有node版本

    • nvm list available 查看可安装的Node JS版本

      建议选择LTS中的版本进行安装（LTS，即Long Term Support，长期支持版本）

    • nvm use + 版本号 切换版本

    • nvm install + 版本号 安装其他版本

    • nvm uninstall + 版本号 卸载安装的某一版本
